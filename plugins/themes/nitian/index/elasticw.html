方案介绍 PROGRAM INTRODUCTION
一款根据用户网站需求，解决用户网站因访问量或资源使用率过高导致的空间稳定性变低，访问速度过慢等问题的方案型产品。多节点负载均衡，为网站提供安全可靠的弹性服务，有效提升网站性能。
方案特性 PROJECT FEATURES
特性 相比较市场同类产品，网站弹性伸缩服务具有以下优势
一键搭建 解决方案自动化部署，一键搭建基础环境，简便快捷即开即用。
安全可靠 多节点负载均衡，避免单点故障，轻松应对网站高并发。
弹性扩展 可按需扩展节点数，无缝业务伸缩，满足网站不同时期的访问需求。
自助管理 自助管理系统，独立操作平台，可直接管理网站，多种管理功能一键操作。

应用场景 APPLICATION SCENARICS

企业类网站

日均并发数平均，要求网站质量高，打开速度快，稳定性高，不受其他因素影响。

门户类网站

海量用户，日均访问量高，存储空间需求大，网站稳定性及后期扩展需求显著。

商城类网站

数据信息多，单点在线率高，同一时间点并发数相对悬殊大，需多节点负载均衡保证网站正常运行。
